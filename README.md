#O verview
This is a nodejs application that can consume JSON documents to be stored in a standalone redis instance.  

It has the capability to:
.Create entry
.Update entry
.Read entry 
.Remove entry

## Required Nodejs Packages
.redis
.express
.bodyparser

# Quick Setup
1. pull this repo
2. docker pull redis
3. docker run -p 6379:6379 redis
4. node app.js
